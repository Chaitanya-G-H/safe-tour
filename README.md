## ⚙️ Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Chaitanya-G-H/safe-tour.git
   ```

2. Navigate to the project folder:

   ```bash
   cd safe-tour
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Run the project:

   ```bash
   npm start
   ``
